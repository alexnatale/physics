Exam #3 Key Equations

Capacitance
Parallel-Plate Capacitor
C = ε₀ A/d
With dielectric: C = κε₀ A/d
Spherical Capacitor
C = 4πε₀(ab)/(b-a)
With dielectric: C = 4πκε₀(ab)/(b-a)
Cylindrical Capacitor
C = 2πε₀L/ln(b/a)
With dielectric: C = 2πκε₀L/ln(b/a)
Capacitors in Series
1/Ceq = 1/C₁ + 1/C₂ + 1/C₃ + ...
Capacitors in Parallel
Ceq = C₁ + C₂ + C₃ + ...
Energy Stored in a Capacitor
U = ½QV = ½CV² = ½Q²/C
Energy Density in Electric Field
u = ½ε₀E²
With dielectric: u = ½κε₀E²
Current and Resistance
Current and Current Density
I = dQ/dt
J = nevd
I = ∫J·dA
Drift Velocity
vd = I/(neA)
n = number density of carriers
Resistance and Resistivity
R = ρL/A
ρ = E/J = m/(ne²τ)
Temperature Dependence
R = R₀[1 + α(T - T₀)]
ρ = ρ₀[1 + α(T - T₀)]
Ohm's Law
V = IR
J = σE
σ = 1/ρ
DC Circuits
Kirchhoff's Rules
Junction rule: ∑Iin = ∑Iout
Loop rule: ∑V = 0
Resistors in Series
Req = R₁ + R₂ + R₃ + ...
Resistors in Parallel
1/Req = 1/R₁ + 1/R₂ + 1/R₃ + ...
Voltage Divider
Vout = (Rout/(R₁ + R₂)) × Vin
Current Divider
I₁ = (R₂/(R₁ + R₂)) × Itotal
Power
P = IV = I²R = V²/R
EMF and Terminal Voltage
Vterminal = ε - Ir
r = internal resistance of source
RC Circuits
Time Constant
τ = RC
Charging a Capacitor
q = C·ε(1 - e-t/τ)
VC = ε(1 - e-t/τ)
I = (ε/R)e-t/τ
Discharging a Capacitor
q = q₀e-t/τ
VC = V₀e-t/τ
I = (V₀/R)e-t/τ
Energy in RC Circuit
U = U₀e-2t/τ
Energy dissipated: U₀(1 - e-2t/τ)
Constants
Elementary charge: e = 1.602 × 10-19 C
Permittivity of free space: ε₀ = 8.85 × 10-12 C²/(N·m²)
Boltzmann constant: k = 1.38 × 10-23 J/K
