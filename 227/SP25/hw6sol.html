Homework 6 — Circuits (Full Solutions)

Problem 1 (Drift Velocity): Solution

The drift speed is given by \[ v_d = \frac{I}{n e A}, \qquad A = \pi r^2. \]

\[ A = \pi(1.5\,\text{mm})^2 = 7.07\times10^{-6}\,\text{m}^2 \]

\[ v_d = \frac{5.0\,\text{A}}{(8.5\times10^{28}\,\text{m}^{-3})(1.602\times10^{-19}\,\text{C})(7.07\times10^{-6}\,\text{m}^2)} \approx 5.2\times10^{-5}\,\text{m\,s}^{-1}. \]

(b) ≈ 0.05 mm s⁻¹, far slower than everyday motions; charge carriers themselves drift very slowly.

Problem 2 (Resistance from Resistivity): Solution

For a uniform cylinder \(R = \rho L / A\).

\[ A = \pi(2.0\,\text{mm})^2 = 1.26\times10^{-5}\,\text{m}^2 \]

\[ R = \frac{2.65\times10^{-8}\,\Omega\,\text{m}\;(2.0\,\text{m})}{1.26\times10^{-5}\,\text{m}^2} \approx 4.2\times10^{-3}\,\Omega. \]

≈ 4.2 mΩ

Problem 3 (Temperature Dependence of Resistivity): Solution

(a) \(\rho_T = \rho_0[1+\alpha(T-T_0)]\).

\[ \rho_{75^{\circ}\text{C}} = 1.68\times10^{-8}[1 + (3.9\times10^{-3})(55)] = 2.04\times10^{-8}\,\Omega\,\text{m}. \]

(b) With \(A = 1.0\,\text{mm}^2 = 1.0\times10^{-6}\,\text{m}^2\),

\[ R_{20^{\circ}\text{C}} = \frac{1.68\times10^{-8}}{1\times10^{-6}} = 0.0168\,\Omega,\quad R_{75^{\circ}\text{C}} = 0.0204\,\Omega. \]

\[ \%\,\Delta R = \frac{0.0204-0.0168}{0.0168}\times100\% \approx 21\%. \]

Problem 4 (Ohm’s Law Check): Solution

Using \(R = V/I\):

  • At 5 V, 0.20 A: \(R = 25\,\Omega\).
  • At 10 V, 0.40 A: \(R = 25\,\Omega\).

The ratio \(V/I\) is constant, so the device obeys Ohm’s law (it is Ohmic).

Problem 5 (Electrical Power & Energy): Solution

(a) \(P = VI = (120\,\text{V})(8.0\,\text{A}) = 9.6\times10^{2}\,\text{W} = 960\,\text{W}.\)

(b) Energy in kilowatt-hours: \(E = (0.960\,\text{kW})(2.5\,\text{h}) = 2.4\,\text{kWh}.\)

Problem 6 (Mesh Analysis): Solution

(a) Clockwise mesh currents \(I_1\) and \(I_2\):

\[ -V_1 + R_1 I_1 + R_3(I_1 - I_2) = 0,\\ -V_2 + R_2 I_2 + R_3(I_2 - I_1) = 0. \]

(b) Substituting values and solving,

\[ I_1 = 0.080\,\text{A},\quad I_2 = 0.060\,\text{A},\quad I_{R_3}=I_1-I_2 = 0.020\,\text{A}. \]

Problem 7 (Power Rating): Solution

(a) Maximum current: \(I_{\max} = \sqrt{P/R} = \sqrt{5.0\,\text{W}/10\,\Omega} = 0.707\,\text{A}.\)

(b) Corresponding voltage: \(V_{\max} = I_{\max}R = 0.707\,\text{A}\times10\,\Omega \approx 7.07\,\text{V}.\)

Problem 8 (Voltage Divider): Solution

\[ V_{R_2} = V\frac{R_2}{R_1+R_2} = 12\,\text{V}\times\frac{200\,\Omega}{100\,\Omega+200\,\Omega} = 8.0\,\text{V}. \]

Problem 9 (Current Divider): Solution

Total current \(I_{\text{tot}} = 2.0\,\text{A}\).

For two resistors in parallel, the branch currents are

\[ I_{R_1} = I_{\text{tot}}\frac{R_2}{R_1+R_2} = 2.0\,\text{A}\times\frac{300\,\Omega}{150\,\Omega+300\,\Omega} = 1.33\,\text{A}, \]

\[ I_{R_2} = I_{\text{tot}}\frac{R_1}{R_1+R_2} = 2.0\,\text{A}\times\frac{150\,\Omega}{150\,\Omega+300\,\Omega} = 0.67\,\text{A}. \]

Problem 10 (RC Circuit Transient Response): Solution

(a) Time constant:

\[ \tau = RC = (1.0\times10^{3}\,\Omega)(10\,\mu\text{F}) = 0.010\,\text{s}. \]

(b) Voltage at \(t = 2\tau\):

\[ V_C(2\tau) = V_\infty\bigl(1 - e^{-2}\bigr) = 12\,\text{V}\bigl(1 - e^{-2}\bigr) \approx 12\,\text{V}(1 - 0.1353) \approx 10.4\,\text{V}. \]
