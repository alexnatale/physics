Problem 1: Velocity Integration & Displacement

  1. (a) Displacement:
    • The velocity is given by: v(t) = 20.0 + 2.0t (m/s).
    • Displacement from t = 0 to t = 10 s is:
      \( \Delta x = \int_0^{10} (20.0 + 2.0t) \, dt \).
    • Integrate:
      \( \int (20.0 + 2.0t) \, dt = 20.0t + t^2 \).
    • Evaluate from 0 to 10:
      \(20.0(10) + (10)^2 = 200 + 100 = 300 \text{ m}\).
  2. (b) Final Velocity:
    • Plug t = 10 into v(t):
      \(v(10) = 20.0 + 2.0(10) = 20.0 + 20.0 = 40.0\) m/s.
  3. (c) Comparison with Average Velocity:
    • The average velocity is \[ v_{\text{avg}} = \frac{v(0) + v(10)}{2} = \frac{20.0 + 40.0}{2} = 30.0 \text{ m/s}. \]
    • Over 10 s, displacement = \(30.0 \text{ m/s} \times 10 \text{ s} = 300 \text{ m}\), which agrees with the integration result.

Problem 2: Free-Fall with Integration

  1. (a) Acceleration:
    • Taking downward as positive, \(a(t) = +9.8\) m/s².
  2. (b) Velocity:
    • Integrate acceleration: \[ v(t) = \int a(t)\, dt = 9.8t + C. \]
    • Given the initial speed \(v(0) = +5.0\) m/s (downward),
      we have \(C = 5.0\). Thus, \(v(t) = 9.8t + 5.0\) m/s.
  3. (c) Position:
    • Define \(y\) as the distance fallen (with \(y = 0\) at the start). Then: \[ y(t) = \int v(t)\, dt = \int (9.8t + 5.0) \, dt = 4.9t^2 + 5.0t. \]
    • Since the object is thrown from a height of 80.0 m above the ground, the ground is reached when \(y(t) = 80.0\) m.
  4. (d) Time to Reach the Ground:
    • Set \(4.9t^2 + 5.0t = 80.0\): \[ 4.9t^2 + 5.0t - 80.0 = 0. \]
    • Solve using the quadratic formula: \[ t = \frac{-5.0 \pm \sqrt{(5.0)^2 - 4(4.9)(-80.0)}}{2(4.9)}. \]
    • Compute the discriminant: \[ 25 + 4 \times 4.9 \times 80 = 25 + 1568 = 1593. \]
    • Taking the positive root: \[ t \approx \frac{-5.0 + \sqrt{1593}}{9.8} \approx \frac{-5.0 + 39.91}{9.8} \approx \frac{34.91}{9.8} \approx 3.56 \text{ s}. \]
  5. (e) Speed at Impact:
    • \(v(3.56) = 9.8(3.56) + 5.0 \approx 34.89 + 5.0 \approx 39.89\) m/s.
    • Thus, the impact speed is approximately 39.9 m/s.

Problem 3: Projectile-Like 1D Upward Throw

  1. (a) Time to Maximum Height:
    • For an upward throw with initial speed 15.0 m/s and acceleration \(a = -9.8\) m/s² (upward positive):
      Set \(v(t) = 15.0 - 9.8t = 0\).
    • Solve for \(t\): \[ t = \frac{15.0}{9.8} \approx 1.53 \text{ s}. \]
  2. (b) Maximum Height:
    • Use the displacement formula: \[ y = v_0t - 4.9t^2. \]
    • At \(t = 1.53\) s: \[ y \approx 15.0(1.53) - 4.9(1.53)^2 \approx 22.95 - 11.45 \approx 11.50 \text{ m}. \]
  3. (c) Total Time in Air:
    • For symmetric projectile motion (ignoring air resistance), time up equals time down.
      Total time \(\approx 2 \times 1.53 \approx 3.06\) s.

Problem 4: Uniform Acceleration & Displacement

  1. (a) Constant Acceleration:
    • \(a = \frac{8.0 - 3.0}{5.0} = \frac{5.0}{5.0} = 1.0 \text{ m/s}^2.\)
  2. (b) Displacement via Integration:
    • The velocity function: \(v(t) = 3.0 + 1.0t\).
    • Integrate: \[ x(t) = \int_0^5 (3.0 + t) \, dt = \left[3.0t + \frac{1}{2}t^2\right]_0^5. \]
    • Evaluate: \[ x(5) = 3.0(5) + 0.5(25) = 15 + 12.5 = 27.5 \text{ m}. \]
  3. (c) Average Velocity Check:
    • \(v_{\text{avg}} = \frac{3.0 + 8.0}{2} = 5.5 \text{ m/s}\).
    • Displacement \(= 5.5 \times 5 = 27.5\) m, which agrees with the integrated value.

Problem 5: Vector Basics

  1. (a) Magnitude of A:
    • \(A = 3.0\hat{i} + 4.0\hat{j}\) has magnitude \(\sqrt{3.0^2 + 4.0^2} = \sqrt{9 + 16} = 5.0\) m.
  2. (b) Angle with the +x-axis:
    • \(\theta = \arctan\left(\frac{4.0}{3.0}\right) \approx 53.13^\circ.\)
  3. (c) Sum with Vector B:
    • Given \(B = -2.0\hat{i} + 1.0\hat{j}\), then \[ A+B = (3.0 + (-2.0))\hat{i} + (4.0 + 1.0)\hat{j} = 1.0\hat{i} + 5.0\hat{j}. \]

Problem 6: Vector Addition & Angles

  1. (a) Components of \(D_1\):
    • \(D_1 = 5.0\) m at 30° above +x:
      \(D_{1x} = 5.0\cos30^\circ \approx 5.0 \times 0.866 \approx 4.33\) m,
      \(D_{1y} = 5.0\sin30^\circ = 5.0 \times 0.5 = 2.5\) m.
  2. (a) Components of \(D_2\):
    • \(D_2 = 7.0\) m at 110° from +x:
      \(D_{2x} = 7.0\cos110^\circ \approx 7.0 \times (-0.342) \approx -2.39\) m,
      \(D_{2y} = 7.0\sin110^\circ \approx 7.0 \times 0.94 \approx 6.58\) m.
  3. (b) Resultant Vector \(D = D_1 + D_2\):
    • \(D_x = 4.33 + (-2.39) \approx 1.94\) m,
      \(D_y = 2.5 + 6.58 \approx 9.08\) m.
  4. (c) Magnitude and Direction of \(D\):
    • Magnitude: \[ |D| = \sqrt{(1.94)^2 + (9.08)^2} \approx \sqrt{3.76 + 82.46} \approx \sqrt{86.22} \approx 9.29 \text{ m}. \]
    • Direction: \[ \theta = \arctan\left(\frac{9.08}{1.94}\right) \approx \arctan(4.68) \approx 78.0^\circ \text{ above the +x-axis}. \]

Problem 7: Upward Launch from a Platform

  1. (a) Equations of Motion:
    • Assume upward as positive; the object is launched from a platform 2.0 m above ground with initial speed 12.0 m/s.
    • Acceleration: \(a = -9.8\) m/s².
    • Velocity: \(v(t) = 12.0 - 9.8t\).
    • Position: \(y(t) = 2.0 + 12.0t - 4.9t^2\) (with \(y(0)=2.0\) m).
  2. (b) Time to Hit the Ground:
    • Set \(y(t)=0\): \[ 2.0 + 12.0t - 4.9t^2 = 0 \quad \Longrightarrow \quad 4.9t^2 - 12.0t - 2.0 = 0. \]
    • Solve using the quadratic formula: \[ t = \frac{12.0 \pm \sqrt{(12.0)^2 + 4(4.9)(2.0)}}{2(4.9)}. \]
    • Discriminant: \(144 + 39.2 = 183.2\); \(\sqrt{183.2} \approx 13.54\).
    • Taking the positive root: \[ t \approx \frac{12.0 + 13.54}{9.8} \approx \frac{25.54}{9.8} \approx 2.61 \text{ s}. \]
  3. (c) Maximum Height Above the Platform:
    • The maximum height is reached when \(v(t)=0\): \[ 12.0 - 9.8t = 0 \quad \Longrightarrow \quad t \approx \frac{12.0}{9.8} \approx 1.22 \text{ s}. \]
    • Plug \(t = 1.22\) s into \(y(t)\): \[ y(1.22) = 2.0 + 12.0(1.22) - 4.9(1.22)^2. \]
    • Calculation:
      \(12.0 \times 1.22 \approx 14.64\);
      \(1.22^2 \approx 1.488\) so \(4.9 \times 1.488 \approx 7.29\);
      \(y(1.22) \approx 2.0 + 14.64 - 7.29 \approx 9.35\) m.
    • Thus, the maximum height above ground is approximately 9.35 m (which is about 7.35 m above the 2.0 m platform).

Problem 8: Position Function & Derivatives

  1. (a) Velocity:
    • Given \(x(t) = 2.0t^2 - 0.5t + 1.0\), differentiate: \[ v(t) = \frac{dx}{dt} = 4.0t - 0.5. \]
  2. (b) Acceleration:
    • Differentiate \(v(t)\): \[ a(t) = \frac{dv}{dt} = 4.0 \text{ m/s}^2. \]
  3. (c) Maximum Velocity:
    • Since \(v(t)=4.0t - 0.5\) is a linear function (with constant positive acceleration), it increases without bound on an open interval.
      Thus, over any finite interval the maximum velocity is attained at the upper endpoint.

Problem 9: Deceleration & Distance

  1. (a) Constant Deceleration:
    • The car decelerates from 30.0 m/s to 0 in 4.0 s: \[ a = \frac{0 - 30.0}{4.0} = -7.5 \text{ m/s}^2. \]
  2. (b) Distance Covered:
    • With \(v(t) = 30.0 - 7.5t\), the displacement is: \[ x = \int_0^4 (30.0 - 7.5t) \, dt. \]
    • Integrate: \[ \int (30.0 - 7.5t) \, dt = 30.0t - 3.75t^2. \]
    • Evaluate from 0 to 4: \[ x = 30.0(4) - 3.75(16) = 120 - 60 = 60 \text{ m}. \]
  3. (c) Average Velocity Check:
    • Average velocity \(=\frac{30.0 + 0}{2} = 15.0\) m/s.
    • Distance \(= 15.0 \times 4.0 = 60\) m, which agrees with the integrated result.

Problem 10: Displacement from Velocity

  1. (a) Find \(x(t)\):
    • Given \(v(t) = 4t - 2\), integrate: \[ x(t) = \int (4t - 2) \, dt = 2t^2 - 2t + C. \]
    • With \(x(0)=0\), we have \(C = 0\). Thus, \(x(t) = 2t^2 - 2t\).
  2. (b) Displacement between \(t=1\) s and \(t=3\) s:
    • \(x(3) = 2(9) - 2(3) = 18 - 6 = 12\) m, and \(x(1) = 2(1) - 2(1) = 0\) m.
    • Displacement \(= 12 - 0 = 12\) m.
  3. (c) Time when \(v(t)=0\):
    • Solve \(4t - 2 = 0 \Rightarrow t = 0.5\) s.
