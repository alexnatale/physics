Problem 1 (Force & Friction): A 25.0 kg crate is initially at rest on a horizontal floor. The coefficient of static friction is 0.40 and the coefficient of kinetic friction is 0.30.

Solution:

(a) Minimum force to start the crate moving:

To overcome static friction, we need:

\(F_{min} = \mu_s \cdot N = \mu_s \cdot mg\)

Where \(N = mg\) (normal force equals weight on horizontal surface)

\(F_{min} = 0.40 \times 25.0 \times 9.8 = 98.0\) N

Minimum force = 98.0 N
(b) Force needed to maintain constant velocity:

For constant velocity, net force = 0, so applied force must equal kinetic friction:

\(F_{constant} = \mu_k \cdot N = \mu_k \cdot mg\)

\(F_{constant} = 0.30 \times 25.0 \times 9.8 = 73.5\) N

Force for constant velocity = 73.5 N
(c) Acceleration with 130 N applied force:

Using Newton's second law:

\(F_{net} = F_{applied} - f_k = ma\)

\(130 - 73.5 = 25.0 \times a\)

\(a = \frac{56.5}{25.0} = 2.26\) m/s²

Acceleration = 2.26 m/s²

Problem 2 (Connected Blocks with Friction): Two blocks with masses m₁ = 5.0 kg and m₂ = 3.0 kg are connected by a light string that passes over a frictionless pulley. Block m₁ sits on a rough horizontal surface with coefficient of kinetic friction μₖ = 0.25, while m₂ hangs freely.

Solution:

(a) Acceleration of the system:

For block m₁ (horizontal): \(T - f_k = m_1a\)

Where \(f_k = \mu_k m_1 g = 0.25 \times 5.0 \times 9.8 = 12.25\) N

For block m₂ (vertical): \(m_2g - T = m_2a\)

Adding the equations:

\(m_2g - f_k = (m_1 + m_2)a\)

\(3.0 \times 9.8 - 12.25 = (5.0 + 3.0)a\)

\(29.4 - 12.25 = 8.0a\)

\(a = \frac{17.15}{8.0} = 2.14\) m/s²

Acceleration = 2.14 m/s²
(b) Tension in the string:

Using the equation for m₂:

\(T = m_2g - m_2a = m_2(g - a)\)

\(T = 3.0(9.8 - 2.14) = 3.0 \times 7.66 = 23.0\) N

Tension = 23.0 N

Problem 3 (Inclined Plane & Friction): A 4.0 kg block slides down a 30° inclined plane. The coefficient of kinetic friction between the block and the plane is 0.20.

Solution:

(a) Component of weight parallel to incline:
\(F_{\parallel} = mg\sin\theta = 4.0 \times 9.8 \times \sin(30°)\)

\(F_{\parallel} = 4.0 \times 9.8 \times 0.5 = 19.6\) N

Parallel component = 19.6 N
(b) Normal force and frictional force:

Normal force:

\(N = mg\cos\theta = 4.0 \times 9.8 \times \cos(30°)\)

\(N = 4.0 \times 9.8 \times 0.866 = 33.9\) N

Frictional force:

\(f_k = \mu_k N = 0.20 \times 33.9 = 6.78\) N
Normal force = 33.9 N, Friction = 6.78 N
(c) Net force and acceleration:
\(F_{net} = F_{\parallel} - f_k = 19.6 - 6.78 = 12.8\) N

\(a = \frac{F_{net}}{m} = \frac{12.8}{4.0} = 3.20\) m/s²

Net force = 12.8 N, Acceleration = 3.20 m/s²
(d) Speed after sliding 3.0 m:

Using kinematics: \(v^2 = v_0^2 + 2as\)

Starting from rest: \(v^2 = 0 + 2(3.20)(3.0) = 19.2\)

\(v = \sqrt{19.2} = 4.38\) m/s

Final speed = 4.38 m/s

Problem 4 (Connected Blocks on Inclined Surfaces): Block A (6.0 kg) rests on a 25° incline. It is connected by a light cord over a frictionless pulley to Block B (4.0 kg) on a 40° incline. Both inclines are frictionless.

Solution:

(a) Acceleration and direction of motion:

Forces parallel to inclines:

Block A: \(F_A = m_A g \sin(25°) = 6.0 \times 9.8 \times 0.4226 = 24.85\) N

Block B: \(F_B = m_B g \sin(40°) = 4.0 \times 9.8 \times 0.6428 = 25.20\) N

Since F_B > F_A, Block B moves down its incline

Net force equation:

\(F_B - F_A = (m_A + m_B)a\)

\(25.20 - 24.85 = (6.0 + 4.0)a\)

\(0.35 = 10.0a\)

\(a = 0.035\) m/s²

Acceleration = 0.035 m/s² (Block A moves up, Block B moves down)
(b) Tension in the cord:

Using Block A equation: \(T - F_A = m_A a\)

\(T = F_A + m_A a = 24.85 + 6.0(0.035)\)

\(T = 24.85 + 0.21 = 25.06\) N

Tension = 25.1 N
(c) Time to reach the pulley:

Using kinematics: \(s = v_0 t + \frac{1}{2}at^2\)

Starting from rest: \(2.0 = 0 + \frac{1}{2}(0.035)t^2\)

\(t^2 = \frac{4.0}{0.035} = 114.3\)

\(t = 10.7\) s

Time = 10.7 s

Problem 5 (Stacked Blocks with External Force): A 5.0 kg block (Block A) sits on top of a 10.0 kg block (Block B), which rests on a horizontal, frictionless surface. The coefficient of static friction between the blocks is 0.35.

Solution:

(a) Maximum force before slipping:

For no slipping, both blocks have same acceleration:

Maximum friction on A: \(f_{s,max} = \mu_s m_A g = 0.35 \times 5.0 \times 9.8 = 17.15\) N

This provides maximum acceleration for A: \(a_{max} = \frac{f_{s,max}}{m_A} = \frac{17.15}{5.0} = 3.43\) m/s²

For the system:

\(F_{max} = (m_A + m_B)a_{max} = (5.0 + 10.0) \times 3.43 = 51.5\) N
Maximum force = 51.5 N
(b) Accelerations if blocks slip:

Kinetic friction: \(f_k = \mu_k m_A g = 0.30 \times 5.0 \times 9.8 = 14.7\) N

Block A: \(a_A = \frac{f_k}{m_A} = \frac{14.7}{5.0} = 2.94\) m/s²

Block B: If F > 51.5 N, say F = 60 N:

\(a_B = \frac{F - f_k}{m_B} = \frac{60 - 14.7}{10.0} = 4.53\) m/s²

Block A: 2.94 m/s², Block B: 4.53 m/s² (for F = 60 N)
(c) Acceleration with F = 40.0 N (no slipping):

Since 40.0 N < 51.5 N, blocks move together:

\(a = \frac{F}{m_A + m_B} = \frac{40.0}{15.0} = 2.67\) m/s²
Acceleration = 2.67 m/s²

Problem 6 (Centripetal Force & Banked Curves): A 1200 kg car travels on a circular banked curve with radius 50.0 m. The curve is banked at an angle of 20°.

Solution:

(a) Speed on frictionless banked curve:

For no friction, the horizontal component of normal force provides centripetal force:

\(v = \sqrt{rg\tan\theta} = \sqrt{50.0 \times 9.8 \times \tan(20°)}\)

\(v = \sqrt{50.0 \times 9.8 \times 0.364} = \sqrt{178.4} = 13.4\) m/s

Speed = 13.4 m/s
(b) Maximum speed with friction:

With friction pointing down the bank:

\(v_{max} = \sqrt{\frac{rg(\sin\theta + \mu_s\cos\theta)}{\cos\theta - \mu_s\sin\theta}}\)

\(v_{max} = \sqrt{\frac{50.0 \times 9.8(0.342 + 0.40 \times 0.940)}{0.940 - 0.40 \times 0.342}}\)

\(v_{max} = \sqrt{\frac{490(0.718)}{0.803}} = \sqrt{438} = 20.9\) m/s

Maximum speed = 20.9 m/s
(c) Minimum speed with friction:

With friction pointing up the bank:

\(v_{min} = \sqrt{\frac{rg(\sin\theta - \mu_s\cos\theta)}{\cos\theta + \mu_s\sin\theta}}\)

\(v_{min} = \sqrt{\frac{50.0 \times 9.8(0.342 - 0.40 \times 0.940)}{0.940 + 0.40 \times 0.342}}\)

Since numerator is negative, minimum speed is 0 m/s

Minimum speed = 0 m/s

Problem 7 (Uniform Circular Motion & Conical Pendulum): A 0.50 kg object is attached to a 1.2 m string and rotates in a horizontal circle. The string makes an angle of 30° with the vertical.

Solution:

(a) Tension in the string:

Vertical equilibrium: \(T\cos\theta = mg\)

\(T = \frac{mg}{\cos\theta} = \frac{0.50 \times 9.8}{\cos(30°)} = \frac{4.9}{0.866} = 5.66\) N
Tension = 5.66 N
(b) Speed of the object:

Radius of circular path: \(r = L\sin\theta = 1.2 \times \sin(30°) = 0.60\) m

Centripetal force: \(T\sin\theta = \frac{mv^2}{r}\)

\(v = \sqrt{\frac{rT\sin\theta}{m}} = \sqrt{\frac{0.60 \times 5.66 \times 0.5}{0.50}} = \sqrt{3.40} = 1.84\) m/s
Speed = 1.84 m/s
(c) Period of rotation:
\(T = \frac{2\pi r}{v} = \frac{2\pi \times 0.60}{1.84} = 2.05\) s
Period = 2.05 s

Problem 8 (Modified Atwood Machine with Incline): A 3.0 kg block (mass m₁) slides without friction on a 35° incline. It is connected by a light string that passes over a frictionless pulley to a hanging 2.5 kg block (mass m₂).

Solution:

(a) Acceleration of the system:

For m₁ on incline: \(T - m_1g\sin\theta = m_1a\)

For m₂ hanging: \(m_2g - T = m_2a\)

Adding equations:

\(m_2g - m_1g\sin\theta = (m_1 + m_2)a\)

\(2.5 \times 9.8 - 3.0 \times 9.8 \times \sin(35°) = (3.0 + 2.5)a\)

\(24.5 - 16.85 = 5.5a\)

\(a = \frac{7.65}{5.5} = 1.39\) m/s²

Acceleration = 1.39 m/s²
(b) Tension in the string:

Using m₂ equation: \(T = m_2(g - a) = 2.5(9.8 - 1.39) = 2.5 \times 8.41 = 21.0\) N

Tension = 21.0 N
(c) Final velocity after 0.80 m descent:

Using \(v^2 = v_0^2 + 2as\) with \(v_0 = 0\):

\(v^2 = 0 + 2(1.39)(0.80) = 2.22\)

\(v = 1.49\) m/s

Final velocity = 1.49 m/s

Problem 9 (Artificial Gravity in a Rotating Space Station): A space station is designed as a rotating cylinder with radius 85.0 m to create artificial gravity through centripetal acceleration.

Solution:

(a) Angular velocity for Earth gravity:

For centripetal acceleration equal to g:

\(a_c = \omega^2 r = g\)

\(\omega = \sqrt{\frac{g}{r}} = \sqrt{\frac{9.8}{85.0}} = 0.339\) rad/s

Angular velocity = 0.339 rad/s
(b) Linear speed at the rim:
\(v = \omega r = 0.339 \times 85.0 = 28.8\) m/s
Linear speed = 28.8 m/s
(c) Period of rotation:
\(T = \frac{2\pi}{\omega} = \frac{2\pi}{0.339} = 18.5\) s
Period = 18.5 s
(d) Scale reading for 70.0 kg astronaut:

The scale reads the normal force, which equals the "artificial weight":

\(F = ma_c = mg = 70.0 \times 9.8 = 686\) N
Scale reading = 686 N

Problem 10 (Multiple Blocks with Different Surfaces): Three blocks with masses m₁ = 4.0 kg, m₂ = 2.0 kg, and m₃ = 3.0 kg are connected by light strings. Block m₁ is on a horizontal surface with coefficient of kinetic friction μ₁ = 0.30, Block m₂ is on a frictionless 25° incline, and Block m₃ hangs vertically.

Solution:

(a) Acceleration of the system:

Forces on each block:

m₁: \(T_1 - f_k = m_1a\) where \(f_k = 0.30 \times 4.0 \times 9.8 = 11.76\) N

m₂: \(T_2 - T_1 - m_2g\sin(25°) = m_2a\)

m₃: \(m_3g - T_2 = m_3a\)

Adding all equations:

\(m_3g - f_k - m_2g\sin(25°) = (m_1 + m_2 + m_3)a\)

\(3.0 \times 9.8 - 11.76 - 2.0 \times 9.8 \times 0.423 = 9.0a\)

\(29.4 - 11.76 - 8.29 = 9.0a\)

\(a = \frac{9.35}{9.0} = 1.04\) m/s²

Acceleration = 1.04 m/s²
(b) Tensions in both strings:

For T₂ (using m₃): \(T_2 = m_3(g - a) = 3.0(9.8 - 1.04) = 26.3\) N

For T₁ (using m₁): \(T_1 = f_k + m_1a = 11.76 + 4.0 \times 1.04 = 15.9\) N

T₁ = 15.9 N, T₂ = 26.3 N
(c) Distance m₃ falls in 2.0 s:

Using \(s = v_0t + \frac{1}{2}at^2\) with \(v_0 = 0\):

\(s = 0 + \frac{1}{2}(1.04)(2.0)^2 = 2.08\) m

Distance = 2.08 m
